# Advent of Code 2025

[![Ruby](https://github.com/Grasseh/adventofcode-2025/actions/workflows/ruby.yml/badge.svg)](https://github.com/Grasseh/adventofcode-2025/actions/workflows/ruby.yml)

Contains my personal solutions to Advent of code problems.

Advent of code is hosted on http://adventofcode.com

## Requirements

Ruby 3.1.x

Minitest

## Usage

`rake run`

Then enter the number of the problem on the prompt

## Tests

`rake test`

## Benchmarking

`rake benchmark`

To output it in the file

`rake benchmark >benchmark.txt`
