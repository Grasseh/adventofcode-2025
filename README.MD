# Advent of Code 2025

[![Actions Status](https://github.com/grasseh/advent-of-code-2025/workflows/Ruby/badge.svg)](https://github.com/grasseh/advent-of-code-2025/actions)

Contains my personal solutions to Advent of code problems.

Advent of code is hosted on http://adventofcode.com

## Requirements

Ruby 3.1.x

Minitest

## Usage

`rake run`

Then enter the number of the problem on the prompt

## Tests

`rake test`

## Benchmarking

`rake benchmark`

To output it in the file

`rake benchmark >benchmark.txt`
